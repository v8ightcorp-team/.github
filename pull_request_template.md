# PR 유형

- [ ] 새로운 기능 추가

- [ ] 치명적인 버그 긴급 수정
- [ ] 문서 수정
- [ ] 주석 추가 및 수정
- [ ] 코드에 영향을 주지 않는 변경사항(오타 수정, 탭 사이즈 변경, 변수명 변경)
- [ ] 코드 리팩토링
- [ ] 빌드 부분 혹은 패키지 매니저 수정
- [ ] 파일 혹은 폴더명 수정
- [ ] 파일 혹은 폴더 삭제
